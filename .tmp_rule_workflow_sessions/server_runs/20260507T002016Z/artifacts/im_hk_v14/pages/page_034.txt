4.7 Cross-currency Netting on MTM Requirement

Favorable MTM / MTM requirement on positions will always be netted off against each other on
its risk currency first. If there is net favorable MTM in one currency, it could be used to offset
against the net MTM requirement in other currencies. The net favorable MTM in original currency
is converted to HKD equivalent by using the exchange rate with haircut applied to mark down
the amount. The net MTM requirement in original currency is converted to HKD equivalent by
using the exchange rate with haircut applied to mark up the amount.
