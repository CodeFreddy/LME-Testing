## Reference Materials

Historical or external source materials should live under `docs/references/`.

Examples:
- `docs/references/ROADMAP.pdf`
- `docs/references/chat_prompt.md`

These files are useful for:
- context,
- comparison,
- historical rationale,
- and traceability of ideas.

They are **not** the active source of truth for repo governance.

Use the main docs in `docs/` for current policy and implementation guidance.  
See `REFERENCE_POLICY.md` and `docs/references/README.md` for the recommended rules.
